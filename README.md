## Setup

```sh
# clone the repo
git clone https://github.com/slerpyyy/sh4der-jockey.git
cd sh4der-jockey

# build and run
cargo run

# install so you can run it from anywhere
cargo install --path .
```

| ⚠️ | Please note that the tool may drop config files in the folder where the executable is located. |
